# Ubertone Exercice 

---

## Exercice statement
A graphical interface displaying a sinusoid graphically, which can be configured in amplitude and frequency by the user.

---

## Getting Started
```bash
git clone git@github.com:leolcde/Ubertone-.git
cd ubertone_exercice

npm install
npm run dev
```
